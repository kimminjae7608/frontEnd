block 요소 태그 : p, ul, li, h1~h6, hr, br, div, table
        ㄴ 태그 사용된 후에 다음 라인으로 줄바꿈이 일어납니다.

inline 요소 태그 : a, img, text format태그(strong, em, ...), span
        ㄴ 태그 사용한 후에 라인을 바꾸지 안습니다.

웹문서 사이의 연결 : a태그
형태 : <a href = " url 또는 해당 폴더의 파일 " target = _black"> </a>
        ㄴ href에 웹문서 연결경로 , target="_black" -> 새탭으로 연결

이미지 넣기 : img태그
형태 : <img src = " 경로 " />
        ㄴ src에 경로를 지정합니다.
        ㄴ 경로 예시 : src="../images/devil shiba dog.png ->.. : 상위폴더

p태그 : 웹문서에 텍스트를 넣어줄 때 사용
        ㄴ사이에 글자 스타일을 태그를 추가할 수 있다.

       - <b> 태그 : 단순히 텍스트를 진하게 표시함.
       - <strong>태그 : 실제 웹문서에서 중요한 강조를 할때 사용함. 스크린리더의 억양이 달라짐. 검색최적화와 웹접근성 효과
       - <i> 태그 : 단순히 이텔릭 체로 표시함.
       - <em> 태그 : 특정 단어에 강세를 두고 발음.
       - <mark> 태그 : 텍스트 하이라이트
       - <del> 태그 : 텍스트 취소선
       - <ins> 태그 : 텍스트 밑줄, 텍스트 강조효과
       - <sup>,<sub>태그 : 위첨자, 아래 첨자
       - <samll> 태그 : 텍스트 사이즈를 작게 표시함.


ul태그 : 순서없는 리스트
        ㄴ ul태그 다음으로 리스트를 추가할 때 li태크 사용
